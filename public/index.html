Yodlr Design Challenge

Hello and welcome to my Yodlr webpage. This was created as of an example of my design skills. This frontend site utilizes jQuery and Bootstrap while the backend uses NodeJS. The backend was provided by others so review the README.md to start the backend services.

If you'd like to see my LinkedIn profile, click my image.

Read on to understand the website and enjoy!

Navigation

Yodlr Logo (Home Page)

This is the home page! This links back to this page.

Registration Page

Anybody can register to be a new user. Duplicates are not verified so be careful when entering this information. This would typically be handled by the backend and return error if any duplicates are found. Click here to learn more of this page.

Admin Page

Here are the full admin right to control this site. You must login with the credentials here. A list of all users is shown. You can sort and even search through the users, helpful when users' list is extensive. Click here to know how to use these features or give it a go yourself.

Admin Log Out

Log out of admin user. This link will only show up if the admin is logged in.

Search Form

Given a search text, the form searches all users which contain the search text in last and first names only and is case-insensitive.

Users List

All users are shown in a list here. If using the search form, only list users which are valid searches.

Categories

Default sorting is by ascending last name however this can be changed to sort all users by a different category by simply selecting a different category header. The categories are:

Last Name First Name Email Status

Sorting

The current sorting criteria is shown as a BLUE button with a chevron next to the category header shown here with ascending order:

First Name

If you desire to change to descending order, select the current category again and the chevron will point down shown here:

First Name

Status

The user status is either "Active" or "Pending". A "Pending" button is next to each user to change their status to "Active" but an "Active" status can not be changed back to "Pending" status.
